## CSS 작성하기
static 폴더를 생성 후 그 안에 앱 이름으로 폴더를 다시 생성하고, 그 안에 css 파일 작성
~~~css
/*polls/static/polls/style.css*/
li a {
    color: green;
}
~~~

html파일에 css파일 추가하기
~~~html
# polls/templates/polls/style.css
{% load static %}
<link rel="stylesheet" href="{% static 'polls/style.css' %}">
~~~
